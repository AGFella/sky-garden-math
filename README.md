# Sky Garden Math

A small bilingual (RU/IT) web game for 3rd graders to practice multiplication and division.
Theme: Sky Garden Rescue. Correct answers power the airship and restore floating islands.

## Features
- Three difficulty levels (easy/medium/hard)
- 10-15 question sessions with progress bar
- Streak bonus scoring
- Gentle hints after 2 wrong attempts
- Russian and Italian UI toggle
- Playful sky theme with floating islands

## Run locally
Open `index.html` in your browser.

## Project structure
- `index.html` UI markup
- `styles.css` visual theme and layout
- `app.js` game logic and localization

## Notes
- Division questions are generated with integer answers only.
- You can adjust question ranges in `app.js`.
