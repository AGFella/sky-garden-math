## Features
- Three difficulty levels with adaptive arithmetic challenge generation.
- 10-question rounds with visual progress, scoring, and streak bonuses.
- Optional timer mode with pause/continue and per-round time tracking.
- Captain Kitten animations, flower growth progression, and celebration effects.
- Hall of Fame with mini view and full overlay view.
- Local score saving with separate records for timer and no-timer modes.
- Multilingual UI support (English, Russian, Italian).

## Notes
- Division tasks always produce integer answers.
- Score data is stored locally in the browser.
